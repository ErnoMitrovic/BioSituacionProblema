Es momento de crear la grfica para comparar el nmero de bases de cada secuencia no alineada. Para fines prcticos, se utilizarn los nombres de las secuencias.

bases <- lapply(not_aligned, tolower)
bases <- lapply(bases, s2c)
bases <- lapply(bases, count, wordsize = 1)
b <- c()
for(el in bases){
  b <- c(b, as.numeric(el))
}
B <- list(
  a = b[seq(from = 1, to = length(b), by = 4)],
  c = b[seq(from = 2, to = length(b), by = 4)],
  g = b[seq(from = 3, to = length(b), by = 4)],
  t = b[seq(from = 4, to = length(b), by = 4)]
)
par(mar = c(5,4,4,10), xpd = T)
colors = colormap(nshades = 20)
barplot(matrix(c(B$a, B$c, B$g, B$t), nrow = 20), beside = TRUE, col = colors, las =2, names.arg = c("A", "C", "G", "T"), ylab = "Frecuencia", xlab = "Bases", main = "Bases nitrogenadas de secuencias")
legend("topright",legend = names(speciesLocus) , col = colors,  pt.cex = 1, pch = 15 , cex = 1, inset = c(-0.4,-0.2))

rm(B, bases, b, colors, el)

# Anlisis de grficas

En la grfica de bases nitrogenadas se puede observar mucha variacin entre los virus de distintas especies. Esto tiene sentido porque son secuencias de virus distinos, pero que comparten algn legado de alineamiento que se explorar ms adelante. Adems de esto, se puede confirmar que las secuencias de los murcielagos, los humanos y los pangolines tienen la cercana ms prxima con respecto al nmero de bases nitrogenadas. El coronavirus analizado en el humano es una de las variantes de SARS-COV-2. Cabe resaltar que todas las secuencias utilizads son de diferentes aos.

Revisando ahora la representacin jerrquica alineada, en la tabla se puede confirmar que el genoma del virus del humano (KU291448), los virus de murcilago, siendo el ms representativo (EF065511), y el virus del pangoln (MW532698) tienen una buena correlacin (Li et al., 2020). Los virus que ms se parecen son los centrales, principalmente la relacin de las variantes de las civetas (Kirtipal et al., 2020). Por otra parte, hay muchos virus que no tienen relacin entre ellos, incluso se puede concluir que hay menos virus que se relacionan entre s que los que conservan una relacin entre ellos.

Finalmente, observando el rbol filogentico se comprueban varias de las relaciones antes mencionadas con la tabla. Se tiene la certeza de que hubo zoonosis que derivan desde los murcilagos como hospedador primario, sin embargo, no se tiene todava la certeza de un nexo directo entre el virus del pangoln con el virus del SARS-COV-2. (Mousavizadeh & Ghasemi, 2020)
